# Magisk

{% hint style="info" %}
Для удаления Magisk необходимо переименовать файл в `uninstall.zip` в TWRP и прошить его.
{% endhint %}

{% hint style="info" %}
Для установки Magisk просто прошейте его в TWRP. Вы можете прошить как \*.apk, так и \*.zip - разницы нет.
{% endhint %}
