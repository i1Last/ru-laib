# %% Работоспособность драйверов на ПК

{% tabs %}
{% tab title="Текст" %}
1. Нажимаем правой кнопкой мыши по иконке «Пуск»
2. Выбираем «Диспетчер устройств»
3. Сравниваем, как на втором скриншоте в правой колонке
